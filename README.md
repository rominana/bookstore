# Bookstore

> React Bookstore with Redux

## Built With

- JavaScript
- React
- Redux
- CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

 - NPM updated
 - Modern web browser
 - Git
### Setup

    git clone https://github.com/rominana/bookstore.git
### Install

    npm install
### Usage

    npm run build
    npm start
